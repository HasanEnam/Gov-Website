Title: Government Service Website (Gov-Website)
Repo: public HTML + PHP website with multiple pages.

What it is:
A multi-page web project built using HTML, CSS and basic PHP that mimics a government information or service website.
It includes common sections such as:

Homepage (index.html)

Registration / signup (signup.html, signup.php)

Contact form (contact.html, contact.php)

Service pages for agriculture/education/health etc.

Tech used:

HTML: Static page structure

CSS: Visual layout and style

PHP: Form handling for signup and contact

Images & assets: Icons/banners for UI content
